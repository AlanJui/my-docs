        # Node.JS 環境建置指引

        # 安裝與設定

        （1）安裝 Node.JS 版本管理套件：n (Node.js Versions Manager)

        cd
curl -L https://git.io/n-install | bash
. ~/.bashrc
        1
        2
        3

        （2）以 Node.JS 版本管理套件，安裝 LTS 版本

        n lts

        （1）安裝軟體套件

        pacman -S yarn
